# vendor-lendor (dynamic)

Node.js + Express backend serving your existing UI from \/public\.

## Run locally
npm install
npm start
# open http://localhost:3000

## API (demo)
GET  /api/health
GET  /api/customers
POST /api/customers
GET  /api/loans
POST /api/loans
GET  /api/payments
POST /api/payments

- Use \ender.yaml\ below for infra-as-code
